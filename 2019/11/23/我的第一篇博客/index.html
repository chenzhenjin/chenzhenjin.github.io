hexo使用教程

初始化hexo

启动hexo

新建一篇博客

打开新建的博客md文件

  • 找到source->_posts
  • 把该文件夹放在hbuilder
  • 用markdown语法书写

    放置本地图片

  • 在hexo的source目录下新建一个pic或者你喜欢的名称目录

    重新整理blog

    部署到github

  • 在new repository 的命名是账号名.github.io
  • 在hexo-blog下安装hexo-deployer-git
  • 修改_config.yml
    • 找到最后的type

  • 执行发布到github命令

    改主题

    • 浏览器url输入github.com/litten/hexo-theme-yilia观看
    • 下载主题
    • 再次修改_config.yml找到theme

      每一次提交前

      致谢

    • 程序羊大佬
